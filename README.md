# 🧪 Ngoc Quynh's Frontend Laboratory

A personal laboratory documenting my frontend journey since my student days. It houses everything I've coded—from the very first custom CSS buttons I learned to build, to complex interactive layouts, UI components, and research. A raw, continuously growing archive of my daily learning and passion for frontend development.

---

## ⚠️ Friendly Warning Before You Enter The Lab

Welcome to my slightly messy "digital mad scientist lab"! 🥼✨

Please note that this is a pure **Experimental Zone**. The main purpose of this repository is to satisfy my curiosity, quickly test out random ideas, and archive my coding memories from when I first started learning.

Therefore:

- The core philosophy here is: **"If it works, it works. We'll refactor it... someday."**
- You will find some really cool snippets, but also some "spaghetti code" from the past that might hurt the feelings of Clean Code enthusiasts.
- Expect to see a wild mix of everything—from flashy hover buttons and complex UI grids to some recent AI prompt engineering research I've been playing around with.

### 💡 How to safely explore this lab:

👉 **DO:** Enjoy the smooth live results on the UI dashboard: **[projects.html](https://ngocquynhpham.github.io/projects.html)** · or land on the **[site home](https://ngocquynhpham.github.io/)** first.
