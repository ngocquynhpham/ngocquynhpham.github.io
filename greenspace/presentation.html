2. Cấu Trúc Mã Nguồn (MVC Tinh Gọn)

Kiến trúc thư mục được tổ chức khoa học, tách biệt Frontend và Backend để dễ dàng bảo trì và mở rộng.

📂 /project-greenspace-root/
  • ├── 📁 admin/ // CMS Quản trị
  • ├── 📄 category_manager.php (QL Danh mục)
    ├── 📄 product_list.php
    ├── 📄 product_add.php
    ├── 📄 blog_manager.php
    ├── 📄 blog_add.php (+ Viết bài mới)
    └── 📄 order_manager.php
  • ├── 📁 config/ // Cấu hình hệ thống
  • ├── 📄 db_connect.php (Kết nối DB)
    └── 📄 session.php
  • ├── 📁 includes/ // Thành phần dùng chung
  • ├── 📄 header.php
    ├── 📄 footer.php
    └── 📄 functions.php
  • ├── 📁 pages/ // Logic hiển thị (View)
  • ├── 📄 home.php
    ├── 📄 category.php (Danh sách SP)
    ├── 📄 product_detail.php
    ├── 📄 blog_list.php (Trang tin tức)
    ├── 📄 blog_detail.php
    └── 📄 cart.php
  • ├── 📁 uploads/ // Media
  • ├── 📁 products/
    └── 📁 blogs/
  • ├── 📄 .htaccess // Rewrite URL (SEO)
  • └── 📄 index.php // Router Chính

🔑 Key Files (Tệp quan trọng)

  • config/db_connect.php: "Trái tim" của hệ thống. Chứa thông tin kết nối Database. Chỉ cần sửa file này khi đổi Hosting.
  • index.php (Router): Nhận mọi yêu cầu từ trình duyệt, gọi file cấu hình và điều hướng đến trang `pages/` tương ứng.
  • .htaccess: Chịu trách nhiệm biến URL xấu (`?id=123`) thành URL chuẩn SEO (`/cay-kim-tien`).

🔄 Luồng dữ liệu (Data Flow)

Ví dụ khi khách xem "Cây Kim Tiền":

  1. Khách truy cập: /cay-kim-tien
  2. .htaccess bắt link, gửi về index.php.
  3. index.php gọi db_connect.php kết nối MySQL.
  4. Hệ thống load pages/product_detail.php.
  5. Trang này `include` header.php và hiển thị dữ liệu cây.
